Get started with HomePi

This API provides access to all of the info you need to manage you Raspberry Pi as well as see some of the information belonging to other peoples Pis. Imagine a world where all the Pis talk, share and just get along! The sample code is a call for getting information about a single configured Pi. To try it out use the example REST client below to send a GET request to /services/homepi/pi/2e848bg934.

To get started managing your Raspberry Pis sign-in with G+

Manage all of your online Raspberry Pis from one location! View logged data, push new software and receive alerts from your Pi no matter where it's located, if it has access to the internet it can call home.
